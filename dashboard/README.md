# Muestra de Dashboard y Reportes

Este proyecto es una muestra simple de cómo quedan mis dashboards y reportes. Está diseñado para demostrar la apariencia y funcionalidad básica de interfaces de usuario para visualización de datos.

## Archivos incluidos

- `index.html`: Página principal del dashboard
- `styles.css`: Estilos CSS para el diseño
- `script.js`: Lógica JavaScript para interactividad

## Cómo ejecutar

1. Asegúrate de tener Python instalado.
2. Ejecuta el servidor local: `python -m http.server 8000`
3. Abre tu navegador y ve a `http://localhost:8000`

Esto iniciará un servidor web local que servirá los archivos del dashboard.
